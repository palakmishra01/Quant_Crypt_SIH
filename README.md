## 🚀 Project Overview

**MailQ** is a first-of-its-kind **quantum-secure communication framework** designed to integrate directly with existing email clients (e.g., Gmail, Outlook). It provides a **multi-layered defense system** that protects sensitive communications from both classical and **post-quantum cyber threats**.

The system bridges the gap between **cutting-edge quantum encryption** and **everyday usability**, ensuring future-proof protection without changing how users interact with their existing email systems.

---

## 🔐 Key Features

- **Hybrid Encryption Framework:**  
  Combines **Post-Quantum Cryptography (Kyber)**, **Quantum Key Distribution (QKD)** simulation via **QuKayDee**, and **AES encryption** to deliver multi-layered security.

- **User-Friendly Security Dial:**  
  Lets users select protection levels (Standard, High, Maximum) without needing cryptographic expertise.

- **Visual & Physical Security:**  
  “Maximum Security” messages are **blurred and PIN-locked**, safeguarding against **shoulder-surfing** and unauthorized on-screen access.

- **Backward Compatibility:**  
  Fully integrates with existing email infrastructure — no disruption to current workflows.

- **Modular & Scalable Design:**  
  Supports migration from simulated to **real QKD networks** as hardware becomes available.

---

## 🧩 Technical Architecture

| Layer | Technology | Purpose |
|-------|-------------|----------|
| **Quantum Key Distribution (QKD)** | QuKayDee Simulator | Simulates quantum key exchange network |
| **Post-Quantum Cryptography (PQC)** | Kyber (Python implementation) | Ensures encryption resistant to quantum attacks |
| **Symmetric Encryption** | AES | Fast and reliable data confidentiality |
| **Key Management** | Custom Python module | Handles secure key generation and rotation |
| **Security Interface** | GUI/CLI integration | Allows user to choose protection level easily |
